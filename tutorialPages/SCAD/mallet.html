Code Walkthrough

Code Color Guide

Keyword     Numbers      Boolean

First Cylinder

Make a cylinder and center it:

cylinder(h = 10,r = 3, center = true);

Rotate

Rotate the cylinder 90 degrees:

rotate([0,90,0]){
cylinder(h = 10,r = 3, center = true);
};

Handle Cylinder

Add another cylinder as the handle cylinder:

(h = 15,r = 2);

Resolution

For finer resolution lower the number; the lower the number the higher resolution:

$fs = 0.4;
